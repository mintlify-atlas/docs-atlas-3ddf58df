title: Quickstart
description: Run a complete Structure-from-Motion reconstruction from Python using PyCOLMAP.
---

This guide walks through a full incremental SfM pipeline using PyCOLMAP: from raw images to a 3D reconstruction.

## Prerequisites

```bash
pip install pycolmap
```

You need a directory of images and a writable output directory.

## End-to-end reconstruction

<Steps>
  <Step title="Set up paths">
    ```python
    import pycolmap
    from pathlib import Path

    image_path = Path("/path/to/images")
    database_path = Path("/path/to/output/database.db")
    output_path = Path("/path/to/output/sparse")

    output_path.mkdir(parents=True, exist_ok=True)
    ```
  </Step>

  <Step title="Extract features">
    Feature extraction reads images, detects keypoints, and computes descriptors. Results are stored in an SQLite database.

    ```python
    pycolmap.extract_features(
        database_path=database_path,
        image_path=image_path,
        camera_mode=pycolmap.CameraMode.AUTO,
        device=pycolmap.Device.auto,
    )
    ```

    Use `pycolmap.Device.cuda` to force GPU extraction, or `pycolmap.Device.cpu` to force CPU.
  </Step>

  <Step title="Match features">
    Match features between image pairs. For small image sets, exhaustive matching works well:

    ```python
    pycolmap.match_exhaustive(
        database_path=database_path,
        device=pycolmap.Device.auto,
    )
    ```

    For larger datasets, use sequential or vocabulary-tree matching:

    ```python
    # Sequential matching (images captured in order)
    pycolmap.match_sequential(database_path=database_path)

    # Vocabulary-tree matching (requires a vocab tree file)
    pycolmap.match_vocabtree(
        database_path=database_path,
        pairing_options=pycolmap.VocabTreePairingOptions(
            vocab_tree_path="/path/to/vocab_tree.bin",
            num_images=50,
        ),
    )
    ```
  </Step>

  <Step title="Run incremental mapping">
    Incremental mapping registers images one at a time and triangulates 3D points:

    ```python
    maps = pycolmap.incremental_mapping(
        database_path=database_path,
        image_path=image_path,
        output_path=output_path,
    )
    ```

    The return value is a dict mapping a reconstruction index to a `pycolmap.Reconstruction` object. Multiple reconstructions may be returned if the scene is disconnected.

    ```python
    print(f"Found {len(maps)} reconstruction(s)")
    reconstruction = maps[0]
    print(reconstruction.summary())
    ```
  </Step>

  <Step title="Inspect the reconstruction">
    ```python
    reconstruction = maps[0]

    # Registered images
    print(f"Registered images: {reconstruction.num_reg_images()}")
    for image_id in reconstruction.reg_image_ids():
        image = reconstruction.image(image_id)
        camera = reconstruction.camera(image.camera_id)
        print(f"  {image.name}: {camera.model_name} {camera.width}x{camera.height}")

    # 3D points
    print(f"Points3D: {reconstruction.num_points3D()}")
    for point3D_id, point3D in reconstruction.points3D.items():
        print(f"  [{point3D_id}] xyz={point3D.xyz}, error={point3D.error:.3f}")
    ```
  </Step>

  <Step title="Export the reconstruction">
    ```python
    # Write in COLMAP binary format (default)
    reconstruction.write(output_path)

    # Write in text format (human-readable)
    reconstruction.write_text(output_path)

    # Export 3D points as PLY point cloud
    reconstruction.export_PLY(output_path / "points3D.ply")
    ```
  </Step>
</Steps>

## Reading an existing reconstruction

Load a reconstruction written by COLMAP or a previous PyCOLMAP run:

```python
import pycolmap
from pathlib import Path

# Reads binary or text format automatically
reconstruction = pycolmap.Reconstruction(Path("/path/to/sparse/0"))

print(reconstruction.summary())

# Access cameras
for camera_id, camera in reconstruction.cameras.items():
    print(f"Camera {camera_id}: {camera.model_name}, f={camera.focal_length:.1f}")

# Access images
for image_id, image in reconstruction.images.items():
    if image.has_pose:
        pose = image.cam_from_world()
        print(f"{image.name}: translation={pose.translation}")

# Find a specific image by name
image = reconstruction.find_image_with_name("frame_0001.jpg")
if image is not None:
    print(f"Found image id={image.image_id}")
```

## Running bundle adjustment

After mapping, you can refine the reconstruction with additional bundle adjustment:

```python
pycolmap.bundle_adjustment(
    reconstruction=reconstruction,
    options=pycolmap.BundleAdjustmentOptions(
        refine_focal_length=True,
        refine_principal_point=False,
        refine_extra_params=True,
    ),
)
```
