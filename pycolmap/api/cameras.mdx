`pycolmap.Camera` stores camera intrinsics: the projection model, focal length, principal point, and any distortion parameters.

## Camera models

PyCOLMAP exposes all COLMAP camera models via `pycolmap.CameraModelId`:

| Model name | ID | Description |
|---|---|---|
| `SIMPLE_PINHOLE` | 0 | f, cx, cy |
| `PINHOLE` | 1 | fx, fy, cx, cy |
| `SIMPLE_RADIAL` | 2 | f, cx, cy, k |
| `RADIAL` | 3 | f, cx, cy, k1, k2 |
| `OPENCV` | 4 | fx, fy, cx, cy, k1, k2, p1, p2 |
| `OPENCV_FISHEYE` | 5 | fx, fy, cx, cy, k1, k2, k3, k4 |
| `FULL_OPENCV` | 6 | fx, fy, cx, cy, k1, k2, p1, p2, k3, k4, k5, k6 |
| `FOV` | 7 | fx, fy, cx, cy, omega |
| `SIMPLE_RADIAL_FISHEYE` | 8 | f, cx, cy, k |
| `RADIAL_FISHEYE` | 9 | f, cx, cy, k1, k2 |
| `THIN_PRISM_FISHEYE` | 10 | fx, fy, cx, cy, k1, k2, p1, p2, k3, k4, sx1, sy1 |
| `RAD_TAN_THIN_PRISM_FISHEYE` | 11 | fx, fy, cx, cy, k1-k6, p1, p2, sx1, sy1 |
| `SIMPLE_DIVISION` | 12 | f, cx, cy, k |
| `DIVISION` | 13 | f, cx, cy, k1, k2 |
| `SIMPLE_FISHEYE` | 14 | f, cx, cy, k |
| `FISHEYE` | 15 | f, cx, cy, k1, k2 |

## Constructing a Camera

```python
import pycolmap

# Default constructor (uninitialized)
camera = pycolmap.Camera()

# From model name
camera = pycolmap.Camera.create_from_model_name(
    camera_id=1,
    model_name="OPENCV",
    focal_length=800.0,
    width=1920,
    height=1080,
)

# From model ID enum
camera = pycolmap.Camera.create_from_model_id(
    camera_id=1,
    model=pycolmap.CameraModelId.OPENCV,
    focal_length=800.0,
    width=1920,
    height=1080,
)

# The model ID enum can also be constructed from a string
model_id = pycolmap.CameraModelId("SIMPLE_PINHOLE")
```

## Properties

```python
camera.camera_id          # int — unique camera identifier
camera.model              # CameraModelId enum
camera.model_name         # str — model name, e.g. "PINHOLE"
camera.width              # int — image width in pixels
camera.height             # int — image height in pixels
camera.params             # numpy array — all camera parameters
camera.has_prior_focal_length  # bool
```

### Focal length

```python
camera.focal_length    # float — single focal length (fx=fy models)
camera.focal_length_x  # float — horizontal focal length
camera.focal_length_y  # float — vertical focal length
camera.mean_focal_length()  # float — mean of fx and fy

# Set focal length
camera.focal_length = 800.0
camera.focal_length_x = 800.0
camera.focal_length_y = 800.0
```

### Principal point

```python
camera.principal_point_x  # float
camera.principal_point_y  # float

# Set principal point
camera.principal_point_x = 960.0
camera.principal_point_y = 540.0
```

### Parameter vector

```python
# Get all parameters as numpy array (view, not copy)
params = camera.params
print(camera.params_info)  # human-readable parameter description

# Modify parameters in-place
camera.params[0] = 850.0

# Set from a list
camera.params = [800.0, 810.0, 960.0, 540.0, 0.1, -0.05, 0.0, 0.0]

# Serialize to comma-separated string
s = camera.params_to_string()

# Parse from comma-separated string
camera.set_params_from_string("800,800,960,540")

# Get parameter indices by group
focal_idxs = camera.focal_length_idxs()      # [0] for SIMPLE_PINHOLE
pp_idxs    = camera.principal_point_idxs()   # [1, 2] for SIMPLE_PINHOLE
extra_idxs = camera.extra_params_idxs()      # distortion parameter indices
```

## Validation

```python
# Check that params vector length matches the model
camera.verify_params()  # -> bool

# Check for numerically implausible parameters
camera.has_bogus_params(
    min_focal_length_ratio=0.1,
    max_focal_length_ratio=10.0,
    max_extra_param=1.0,
)  # -> bool

# Check whether all extra (distortion) params are zero
camera.is_undistorted()  # -> bool
```

## Projection

### World to image (project)

Project from normalized camera coordinates `(u, v, w)` to pixel coordinates.

```python
import numpy as np

# Single point: returns Optional[np.ndarray] shape (2,)
image_pt = camera.img_from_cam(np.array([0.1, 0.2, 1.0]))

# Batch: cam_points shape (N, 3), returns list of (2,) arrays
cam_pts = np.array([[0.1, 0.2, 1.0], [0.3, -0.1, 1.0]])
image_pts = camera.img_from_cam(cam_pts)
```

### Image to camera (unproject)

Lift pixel coordinates to normalized camera coordinates.

```python
# Single point: returns Optional[np.ndarray] shape (2,)
cam_pt = camera.cam_from_img(np.array([960.0, 540.0]))

# Batch: image_points shape (N, 2), returns list of (2,) arrays
image_pts = np.array([[960.0, 540.0], [100.0, 200.0]])
cam_pts = camera.cam_from_img(image_pts)

# Batch from Point2DList
cam_pts = camera.cam_from_img(image.points2D)

# Convert a pixel threshold to normalized space
threshold_normalized = camera.cam_from_img_threshold(threshold=1.5)
```

## Calibration matrix

```python
# Returns the 3x3 calibration matrix K
K = camera.calibration_matrix()  # numpy ndarray (3, 3)
```

## Rescaling

```python
# Rescale to new dimensions (adjusts focal length and principal point)
camera.rescale(new_width=960, new_height=540)

# Rescale by a factor
camera.rescale(scale=0.5)
```
