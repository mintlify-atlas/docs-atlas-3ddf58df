---
title: Reconstruction
description: API reference for pycolmap.Reconstruction — the top-level container for cameras, images, and 3D points in a COLMAP scene.
---

`pycolmap.Reconstruction` is the central data structure in PyCOLMAP. It holds the complete scene graph: cameras, images (with their poses), and triangulated 3D points.

## Constructor

```python
# Empty reconstruction
reconstruction = pycolmap.Reconstruction()

# Copy an existing reconstruction
copy = pycolmap.Reconstruction(reconstruction)

# Read from disk (binary or text format, auto-detected)
reconstruction = pycolmap.Reconstruction("/path/to/sparse/0")
```

## Reading and writing

```python
# Read — prefers binary format
reconstruction.read("/path/to/sparse/0")

# Read specific formats
reconstruction.read_binary("/path/to/sparse/0")
reconstruction.read_text("/path/to/sparse/0")

# Write binary format (recommended)
reconstruction.write("/path/to/output")

# Write specific formats
reconstruction.write_binary("/path/to/output")
reconstruction.write_text("/path/to/output")

# Export 3D points as PLY
reconstruction.export_PLY("/path/to/output/points3D.ply")

# Import points from PLY (for visualization only)
reconstruction.import_PLY("/path/to/points.ply")
```

## Properties

### Collections

All collections are returned as references (modifications affect the reconstruction directly).

```python
# Dict[camera_id, Camera]
cameras = reconstruction.cameras

# Dict[image_id, Image]
images = reconstruction.images

# Dict[point3D_id, Point3D]
points3D = reconstruction.points3D

# Dict[rig_id, Rig]
rigs = reconstruction.rigs

# Dict[frame_id, Frame]
frames = reconstruction.frames
```

### Counts

```python
reconstruction.num_cameras()         # Total cameras
reconstruction.num_images()          # Total images
reconstruction.num_reg_images()      # Registered (posed) images
reconstruction.num_points3D()        # Triangulated 3D points
reconstruction.num_rigs()            # Camera rigs
reconstruction.num_frames()          # Frames
reconstruction.num_reg_frames()      # Registered frames
```

### Registered image and frame IDs

```python
# Returns a list of image IDs with a registered pose
reg_image_ids = reconstruction.reg_image_ids()

# Returns a list of registered frame IDs
reg_frame_ids = reconstruction.reg_frame_ids()

# Returns a set of all point3D IDs
point3D_ids = reconstruction.point3D_ids()
```

## Direct accessors

```python
camera  = reconstruction.camera(camera_id)    # raises if not found
image   = reconstruction.image(image_id)      # raises if not found
point3D = reconstruction.point3D(point3D_id)  # raises if not found
rig     = reconstruction.rig(rig_id)
frame   = reconstruction.frame(frame_id)
```

### Existence checks

```python
reconstruction.exists_camera(camera_id)    # -> bool
reconstruction.exists_image(image_id)      # -> bool
reconstruction.exists_point3D(point3D_id)  # -> bool
reconstruction.exists_rig(rig_id)          # -> bool
reconstruction.exists_frame(frame_id)      # -> bool
```

## Adding elements

```python
import pycolmap

# Add a camera
camera = pycolmap.Camera.create_from_model_name(
    camera_id=1,
    model_name="PINHOLE",
    focal_length=1000.0,
    width=1920,
    height=1080,
)
reconstruction.add_camera(camera)

# Add camera and create a matching trivial rig automatically
reconstruction.add_camera_with_trivial_rig(camera)

# Add an image (its camera must already exist)
image = pycolmap.Image(name="frame_0001.jpg", camera_id=1)
reconstruction.add_image(image)

# Add image with a trivial frame (frame_id == image_id)
reconstruction.add_image_with_trivial_frame(image)

# Add image with a trivial frame and register it with a pose
reconstruction.add_image_with_trivial_frame(image, cam_from_world)

# Add a 3D point and return its ID
import numpy as np
track = pycolmap.Track()  # empty track
point3D_id = reconstruction.add_point3D(
    xyz=np.array([1.0, 2.0, 3.0]),
    track=track,
    color=np.array([128, 255, 0], dtype=np.uint8),
)
```

## Coordinate transforms

```python
# Apply a Sim3d similarity transform to all cameras and points
from pycolmap import Sim3d
transform = Sim3d(...)  # scale, rotation, translation
reconstruction.transform(new_from_old_world=transform)

# Normalize the scene (translate centroid to origin, scale to unit extent)
reconstruction.normalize(
    fixed_scale=False,
    extent=10.0,
    min_percentile=0.1,
    max_percentile=0.9,
    use_images=True,
)

# Compute centroid of camera centers or 3D points
centroid = reconstruction.compute_centroid(min_percentile=0.0, max_percentile=1.0)

# Compute axis-aligned bounding box
bbox = reconstruction.compute_bounding_box()

# Crop to a bounding box
reconstruction.crop(bbox)
```

## Editing points and observations

```python
# Merge two 3D points; returns the new point3D_id
new_id = reconstruction.merge_points3D(point3D_id1, point3D_id2)

# Delete a 3D point and all references in images
reconstruction.delete_point3D(point3D_id)

# Delete one observation (and the 3D point if only two observations remain)
reconstruction.delete_observation(image_id, point2D_idx)

# Add an observation to an existing 3D point
track_element = pycolmap.TrackElement(image_id=1, point2D_idx=5)
reconstruction.add_observation(point3D_id, track_element)

# Delete all 2D and 3D points
reconstruction.delete_all_points2D_and_points3D()

# Recompute reprojection errors for all 3D points
reconstruction.update_point_3d_errors()
```
